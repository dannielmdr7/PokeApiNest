# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar el comando

```
yarn install
```

3. Tener NestJs intalado รณ ejecutar:

```
npm i -g @nestjs/cli
```

4. Levantar la base de Datos.

```
docker-compose up -d
```

5. Reocnstuir la base de Datos.

```
hacer get a este endpont /api/v2/seed
```
